Skill Tree (Global Concept) 🌟
------------------------------------------------
A branching progression system with multiple tiers, where each node represents a learned ability, passive bonus, or specialization. Trees grow through repeated actions, environmental influence, and character identity.

Three parallel trees:
Task Tree - grows from daily habits and consistency
Profession Tree - grows from different jobs
Class Tree - grows from combat, exploration, and class‑specific actions

How the Three Trees Interact:

Task Tree → Profession Tree:  
Discipline boosts crafting efficiency.

Profession Tree → Class Tree:  
Your craft influences your combat stats.

Class Tree → Task Tree:  
Combat perks make certain tasks easier (strength helps with cleaning, agility helps with errands).

Tiered structure:  
Early 0 - 10 → Mid 11 - 20 → Advanced 21 - 30 → Epic 31 - 40 → Legendary 41 - 50

Cross‑tree power:  
Nodes in one tree can enhance nodes in another (e.g., Profession → Class power).


Task Skill Tree (Daily Life → Power)  🌟
----------------------------------------------------------------
This tree rewards consistency, discipline, and routine. It’s the “life‑to‑power” pipeline.

What grows it:
Completing daily tasks
Streaks
Task difficulty
Upgraded buildings (Mines, Library, Workshop)

Node themes:
Efficiency - faster stamina regen, reduced cooldowns
Focus - boosts to Intelligence, Resilience, or Charisma
Momentum - bonuses for streaks or completing multiple tasks in a row
Lifestyle perks - cooking, crafting, gardening, meditation
Self‑care perks - healing bonuses, stress resistance
Exploration prep - pack weight, travel speed, map clarity

Special mechanics:
Streak branches: unlock only after X days of consistency
Category mastery: completing many tasks in one category unlocks a mini‑specialization
Habit mutations: long‑term habits evolve into powerful passives


Profession Skill Tree (Your Trade → Mastery)  🌟
----------------------------------------------------------------
This tree is tied to your professions.

What grows it:
Doing profession tasks
Using profession tools
Selling or completing profession outputs
Visiting profession buildings
Finding rare materials
Learning from NPC/Player mentors

Node themes:
Crafting mastery - better quality, faster production
Resource efficiency - fewer materials needed
Discovery - new recipes, blueprints, formulas
Tool specialization - unique abilities tied to your craft
Economic perks - better prices, trade bonuses
Profession‑combat crossover - e.g., Engineer traps, Chef buffs, Herbalist poisons

Special mechanics:
Epic teachers: rare NPCs who unlock unique branches
Material affinity: using certain materials repeatedly unlocks themed perks
Workshop upgrades: buildings unlock new tiers of the tree


Class Skill Tree (Combat Identity → Character Growth)  🌟
--------------------------------------------------------------------------------
This is the “RPG class fantasy” tree. It defines how you fight, survive, and express your archetype.

What grows it:
Combat actions
Monster kills
Expeditions
Class‑specific tasks
Region exposure (Swamp, Volcano, Island, Forest, Desert)
Class trainers

Primary Elements : 
Crux
Earth
Plant
Water
Air
Fire
Storm 

Base variants : 
Decay
Nano
Portal
Solar
Void
Mech 


Node themes:
Core abilities: attacks, spells, stances
Stat scaling: Strength, Agility, Intelligence, etc.
Signature moves: ultimate abilities, transformations
Damage types: fire, frost, poison, shadow, holy
Defensive styles: shields, dodges, barriers, stealth
Movement: dashes, teleports, leaps

Class‑region combo ideas:
Swamp Mage → poison, decay, illusions
Volcano Warrior → fire, rage, armor‑break
Island Druid → tides, wind, healing

Special mechanics:
Multi‑classing: unlocks hybrid branches
Region imprinting: spending time in a region shifts your class flavor
Monster essence: rare drops unlock unique nodes
Class trials: mini‑quests that unlock new tiers
